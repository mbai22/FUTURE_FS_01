# Assets Directory

Ce dossier contient toutes les images et ressources visuelles utilisées dans le portfolio.

## Images requises

### Images de profil
- **profile.jpg** - Photo de profil professionnelle (400x400px recommandé)

### Images des projets
- **project1.jpg** - Capture d'écran du Mini CRM (400x300px recommandé)
- **project2.jpg** - Capture d'écran du Site Restaurant (400x300px recommandé)  
- **project3.jpg** - Capture d'écran de l'API REST (400x300px recommandé)

### Images pour réseaux sociaux
- **preview.jpg** - Image preview pour Open Graph (1200x630px recommandé)
- **favicon.svg** - Icône du site (format SVG, 32x32px recommandé)

## Instructions

1. **Remplacer les images par défaut** par vos propres captures d'écran
2. **Optimiser les images** pour le web (compression sans perte de qualité)
3. **Maintenir les noms de fichiers** exacts pour éviter les liens cassés
4. **Utiliser des formats modernes** : WebP pour les photos, SVG pour les icônes

## Recommandations

- **Photo de profil** : Fond neutre, visage clairement visible
- **Captures d'écran** : Haute qualité, montrant l'interface principale
- **Preview sociale** : Inclure logo, titre et description du portfolio
